### Running the code

Run files as modules while in root directory.

        python -m Trainers.train_student
        python -m Trainers.train_teacher

### Acknowledegment

- The codebase is a refactored version of the codebase of [LILA-BOTI : Leveraging Isolated Letter Accumulations By Ordering Teacher Insights for Bangla Handwriting Recognition](https://github.com/apurba-nsu-rnd-lab/lila-boti)

* The aim of our project is to get a better understanding of the usage of Deep Learning for Bangla Handwriting Recognition through replicating an existing research paper.
